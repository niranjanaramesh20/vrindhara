# Vrindhara 🌸

A front-end clone of **Vrindhara** — an online jewellery store — built with plain HTML, CSS, and JavaScript.

## 📸 Preview

A responsive product listing page featuring a search bar, filter row, product cards with hover effects, and a footer with policy links.

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

## ✨ Features

- **Live search** — filters product cards in real time as you type
- **Responsive grid layout** — product cards adapt to any screen size using CSS Grid
- **Hover animations** — cards scale and show a shadow on hover
- **Navigation bar** — links to Home, All Products, Categories, Contact Us, and About
- **Footer** — includes policy links and copyright info

## 📁 Project Structure

```
vrindhara-clone/
├── index.html    # Main HTML page
├── style.css     # Styles and layout
└── app.js        # Search/filter functionality
```

## 🚀 Getting Started

No installations or dependencies needed. Just clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/YOUR_USERNAME/vrindhara-clone.git
cd vrindhara-clone
open index.html
```
